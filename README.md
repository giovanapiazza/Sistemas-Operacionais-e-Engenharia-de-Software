# Sistemas-Operacionais-e-Engenharia-de-Software
Este simulador de gerenciamento de memória em Java integra Engenharia de Software e Sistemas Operacionais. Implementa alocação dinâmica (sequencial/paralela) em uma Heap de vetores, usando semáforos para sincronismo de threads e lógica RANDOM para liberação de 30% do espaço. O projeto segue o framework Scrum em 10 sprints com rotação de papéis.
